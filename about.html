Our Project

Hawlucha Helper is a website that allows users to search for information about different Pokemon. Users can search for Pokemon by name or by their Pokedex number. Our website returns said Pokemon's name, Pokedex number, and habitat. Our website retrieves this data from PokeAPI and displays it on an image of a Pokedex. Our website also uses another API to display images of the first 151 Pokemon.

We used Media Queries to make the website display differently based on the user's screen size. Screen sizes 600 pixels or less are mobile devices, and screens 600 pixels or greater are desktop, laptop, or tablet devices.
